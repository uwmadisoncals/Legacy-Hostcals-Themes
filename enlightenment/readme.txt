== Description ==

Our flagship multipurpose theme powered by Enlightenment Framework.
With a clean, modern and responsive design backed by a powerful and flexible code base.
Showcase any type of content anywhere. Create unlimited layouts.
Customize all templates to your preference. Take WordPress as CMS to another level.

== Installation ==

Manual installation:

1. Upload the `enlightenment` folder to the `/wp-content/themes/` directory

Installation using "Add New Theme"

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for 'enlightenment'
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Theme Options to change theme specific options

== Theme Notes ==

=== Drop-down Menus ===

Standard menu items support drop-down menus that are one level deep.
Menu items with the class `mega-menu` support submenus two levels deep
but are shown in the same drop-down.
